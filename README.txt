YouTube動画をURLから簡単にダウンロードできるアプリです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📱 起動方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【Macの場合】
  1. 「start_mac.command」をダブルクリック
  2. 初回は「開発元を検証できないため...」と表示される場合があります
     → 「システム設定」→「プライバシーとセキュリティ」→「このまま開く」
  3. ブラウザが自動で開きます

【Windowsの場合】
  1. 「start_windows.bat」をダブルクリック
  2. 「WindowsによってPCが保護されました」と表示される場合
     → 「詳細情報」→「実行」をクリック
  3. ブラウザが自動で開きます


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 ダウンロード方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. YouTube動画のURLをコピー
  2. アプリの入力欄に貼り付け（📋ボタンでも貼り付け可能）
  3. 「動画(MP4)」か「音声のみ(MP3)」を選択
  4. 「ダウンロード開始」ボタンをクリック
  5. ダウンロード完了を待つ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📁 保存先
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  【Mac】    ~/Downloads/YouTube/
  【Windows】C:\Users\ユーザー名\Downloads\YouTube\


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚙️ 必要なもの（初回のみ自動インストール）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ・Python 3.8以上

  【Macの場合】
    ターミナルで以下を実行:
    brew install python3

  【Windowsの場合】
    以下からダウンロードしてインストール:
    https://www.python.org/downloads/
    ※インストール時に「Add Python to PATH」にチェック！

Q: 「Pythonがインストールされていません」と表示される
A: 上記「必要なもの」を参考にPythonをインストールしてください

Q: ダウンロードが途中で止まる
A: ネットワーク接続を確認してください。一度アプリを再起動してみてください

Q: 動画が見つからないエラーが出る
A: URLが正しいか確認してください。一部の動画は地域制限でダウンロードできません

Q: Macで「開発元を検証できない」と表示される
A: システム設定 → プライバシーとセキュリティ → 「このまま開く」
